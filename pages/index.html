Top Control Panel

Reading Mode - Turns the background dark and enables text highlighting.
Line Mode/Sentence Mode - Click to toggle between highlighting line by line and highlighting sentence by sentence.
New Input - Click to erase current input and display a text box to paste input into.
Help - Display this dialog box. (Close by clicking "X" in upper right corner)

Reeder Controls

Press Forward or Back at the bottom of the screen to advance/pull back highlighting by one line or sentence.
You may also use the Left and Right arrow keys to do the same.
You may also use the "j" and "k" to do the same (turbo 1337 only).

Adding Text

While browsing any page, highlight the text you want to read and right click, and then select Reedy > Open selection as new text in Reedy.
If you wish to add to the already existing text, right click, and then select Reedy > Add to existing text in Reedy.
